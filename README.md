# The Candor Project — Website
 
Public website for [The Candor Project](https://thecandorproject.org), a civic reform initiative proposing a professional accountability framework for federal politicians and media outlets around factual accuracy.
 
## About the Project
 
The Candor Project argues that the same professional standards applied to doctors, lawyers, and financial advisors should apply to federal officeholders and licensed broadcasters. The framework proposes a two-track accountability mechanism — one for politicians, one for the media — governed by an independent epistemic oversight body with a narrow mandate focused exclusively on empirically falsifiable claims.
 
For the full proposal, see the white paper at [thecandorproject.org](https://thecandorproject.org).
